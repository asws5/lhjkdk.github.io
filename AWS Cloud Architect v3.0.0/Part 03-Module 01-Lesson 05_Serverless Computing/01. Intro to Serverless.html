01. Intro to Serverless

ND320 C2 L3 01 Serverless Intro

Lesson Overview

Serverless intro recap

Key Points

  • Serverless computing offers a platform for developers to run their applications as a series of functions as opposed to a deployable software package.

  • Serverless architecture works best for lightweight applications that have short running processes and experience bursts of activity with lulls in between.

  • Applications with long running and consistent workloads would run most efficiently on an optimized EC2 instance

In this lesson we'll talk about:

  • Cloud Migration
  • AWS Lambda Serverless Costs
  • API Gateway
  • Containers
