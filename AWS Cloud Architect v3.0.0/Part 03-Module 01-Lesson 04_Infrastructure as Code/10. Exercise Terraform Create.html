10. Exercise: Terraform Create

TC Exercise Intro

Creating AWS Infrastructure with Terraform

For this exercise, we will create EC2 instances by writing a configuration file in a code editor declaring the infrastructure we want to create.

We will instruct Terraform to analyze our configuration plan and then create our desired infrastructure by executing the plan.

Finally, after we have successfully provisioned infrastructure with Terraform, we will return to the terminal and destroy the infrastructure with Terraform.

Use Terraform to create and destroy infrastructure with these parameters:

  • A provider block (provider, access key, secret key, and region)
  • A resource block ( ami, at least 2 EC2 instances, instance type, a tag (key/value))

AWS Resource Warning!

This exercise will use resources that are outside of the AWS Free Tier. Do not forget to destroy the EC2 instances when you have completed the exercise.
