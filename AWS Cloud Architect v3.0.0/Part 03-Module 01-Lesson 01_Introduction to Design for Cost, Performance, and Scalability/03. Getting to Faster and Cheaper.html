Performance Efficiency Glossary Terms

New Terms

Term Definition
Baseline Measurement of conditions at the start of your project used to make planning decisions
Cloud Computing Storing and accessing data and applications over the Internet instead of a local or networked computer's hard drive
Infrastructure as Code The process of provisioning and managing your cloud resources by writing a template file that is both human-readable, and machine-consumable
Serverless Compute model where the developer doesn’t need to be concerned with the server because the cloud provider takes care of it

AWS Efficiency recap

Key Points

  • Moving to the cloud doesn’t guarantee that your application will be faster and cost less to operate. In fact, without the proper baseline metrics, you won’t know where to start when planning your move.

  • Most performance issues can be traced to the application itself and some applications will need to be redesigned to be cloud native in order to achieve truly optimized performance. Common causes of performance issues:

    • Poorly designed applications
    • Database design constraints
    • Inefficient network routes

  • Moving to the cloud should improve network response times. Global edge cache locations can bring the server closer to the end user.

Free Response Quiz

QUESTION:

Reflect

What do you think is the best service cloud providers offer their customers?

ANSWER:

I think the best service cloud providers offer is elasticity. Cloud providers offer the ability to automate resource scaling, which is a huge advantage over the data center model. Having your applications configured to autoscale offers peace of mind along with the knowledge that your applications will always have the compute resources they need. You can also expect predictable costs when you use good resource planning practices and use AWS pricing optimization options when configuring your scaling policies.

High Performance Computing Recap

New Terms

Term Definition
High Performance Computing (a.k.a. supercomputing) Supercomputing harnesses the computing power of multiple computers and aggregates them to tackle complex problems in business and academia

Key Points

  • Supercomputing harnesses the computing power of multiple computers and aggregates them to tackle complex problems in business and academia.

  • In supercomputing, computers are called nodes and nodes are organized into groups called clusters.

  • Supercomputing is the ultimate in high performance computing in the cloud.
