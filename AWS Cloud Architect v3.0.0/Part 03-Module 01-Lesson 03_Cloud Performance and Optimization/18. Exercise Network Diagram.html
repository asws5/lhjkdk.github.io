Creating an Architectural Diagram

Please feel free to create your schematic with your preferred diagramming application. I used LucidChart to create the diagrams for this course.

LucidChart has a free version that allows you to save some files without the monthly service charge. The free version should be sufficient for the exercises in this course.

LucidChart

When you create a new blank document, make sure you find and select the AWS 2017 or AWS 2019 shapes by clicking on the gear icon in the upper left corner of the ‘Shapes’ menu and searching for AWS. The AWS shapes will help you create your diagram with accurate shapes and icons.
