09. Solution: Creating an EBS Snapshot

Resizing Exercise Solution heading

My Solution

EBS Snapshot Exercise Solution

Solution Discussion

Key Points

  • EBS snapshots are incremental copies of data, meaning only new/different blocks of data that have changed since the last snapshot are captured in the next snapshot.

  • EBS Snapshots are stored in S3, but not in a user visible bucket. The bucket can be accessed via the EC2 CLI commands

  • Snapshots can be used to create new volumes, create larger volumes, and create volumes in different availability zones
