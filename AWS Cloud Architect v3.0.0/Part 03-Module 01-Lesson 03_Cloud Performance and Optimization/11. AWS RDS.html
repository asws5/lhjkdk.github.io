Steps to Create an RDS Database

Steps to Create an Aurora RDS Database Cluster

  1. Login to the AWS Management console
  2. Search for RDS or browse to RDS in the Services console under Databases
  3. On the AWS RDS Management console, select Create Database
  4. Select Amazon Aurora (mySQL compatibility) as your RDS Database
  5. Name your DB cluster
  6. Configure your DB engine, instance class and storage
  7. Select admin password option
  8. Specify optional cluster settings
  9. Create Database

Read Replica

Read Replica new terms

New Term

Term Definition
Read Replica A copy of the master db that reflects changes to the master instance in almost real time.

Read Replica recap

Read Replica

  • Read-only connections reduce read-traffic to the source database
  • Allows source database to carry out other operations
  • Good for business intelligence and financial engines
  • Can be promoted in disaster recovery

Steps to Create a Read Replica

  1. Login to the AWS Management Console
  2. Browse to the RDS Management page
  3. Select ‘Databases’ in the navigation menu
  4. Select the radio button next to the database cluster where you want to create the reader
  5. Under the ‘Actions’ menu, select Add Reader
  6. On the Add Reader page, configure reader options
  7. Create Reader
