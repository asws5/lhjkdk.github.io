Using Server-Side Encryption Exercise Instructions

Exercise: Using Server-Side Encryption

In this exercise, you will use the AWS API to write a file to an S3 bucket. You can either use the AWS API for a language you're comfortable, or you can use the AWS CLI.

1. Create a Sample Application

Refer to the bottom of this link for guides to AWS SDK documentation for specific languages and the CLI.

When writing your put-object code or command, make sure to choose a specific type of server-side encryption.

2. Run the Code

Run your code to write a file of your choice to an S3 bucket in your AWS account.

3. Verification

1) Open the console and go to the S3 service page.

2) Find the new file that you have uploaded.

3) Select the file and go to the properties tab.

4) Select encryption. The panel will open up and you can verify that the type of encryption selected is indeed what you had specified when writing the file.
