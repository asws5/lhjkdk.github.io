13. Optimizing for Cost

Optimizing Header

Intro to Cost Optimization

ND063 C2 L1 16 Optimize Your Environment

AWS has services to help you

Key Points

  • Establish baselines, monitor peak periods and autoscale based on peaks
  • Use spot instances strategically to reduce costs
  • Private IPs allow you to transfer data for free within the same availability
  • Use consolidated billing to save money if your organization has multiple AWS accounts

Finding the Cost/Performance Sweet Spot

  • Optimization takes time and skill
  • You will probably need to make many small changes
  • Don't underestimate the role of monitoring

With time and experience, it becomes easier to detect areas where optimization is required.
