10. Solution: S3 Lifecycle Policy Configuration

Solution Header

S3 Lifecycle Policy Configuration Solution

ND063 C2 L1 11 S3 Lifecycle Policy

Solution video Recap

My Solution

Your solution might be different, but here is what I came up with:

  • Store the media files in S3 standard for 30 days, then move files to S3 IA
  • Store the media files in S3 IA for 60 days, then move to S3 Glacier and pay AWS for provisioned expedited retrieval if necessary
  • Store the media in S3 Glacier for the remainder of the 180 days, then delete per the lifecycle policy

Lifecycle Policy Rules

Based on what you learned in the S3 Lifecycle exercise, how many days does an object have to be stored in S3 Standard storage before it can transition to another S3 storage tier?
