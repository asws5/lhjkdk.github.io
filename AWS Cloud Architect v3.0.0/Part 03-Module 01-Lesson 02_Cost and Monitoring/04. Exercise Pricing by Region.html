04. Exercise: Pricing by Region

Exercise intro

Calculate Infrastructure Cost in Six Regions

AWS prices services differently in each region. This exercise illustrates how region affects the pricing of various services. We will use the AWS Pricing Calculator to estimate the monthly cost of the exact same services in six regions:

  • N. Virginia
  • N. California
  • Oregon
  • Paris
  • Ireland
  • Mumbai

Price out this group of services:

  • EC2
    • 8 instances
    • Linux operating system
    • c5.xlarge
    • On-Demand Instances
    • General purpose SSD
    • 30 GB storage
  • Elastic Load Balancing
    • 1 Application Load balancer
    • Average New Connections per ALB: 10
    • Average Number of Requests per second per ALB: 10
    • Average Connection Duration: 1 second
  • S3 Storage
    • 4TB
    • No data transfer
  • Database
    • 1 Amazon Aurora Serverless (MySQL Compatible)
    • 4GB RAM (2 Aurora Capacity Units per hour)
    • 100GB Database Storage

Make a note of the estimates and rank the regions below from lowest to highest cost. You can use this Costs by Region Worksheet to record your results.
