Course Outline Summary

Course Outline

This course focuses on architectural best practices around cloud security.

Identity and Access Management

  • We will start with looking at identity and access management in AWS.
  • You will learn to secure users and roles who manage and control resources in the cloud.
  • We will also investigate best practices around authentication, authorization and fine tuning user permissions.

Securing Access to Cloud Infrastructure

  • We will also explore methods to ensure cloud infrastructure and networks are protected from unauthorized access.
  • This will include techniques to secure and harden access to instances and servers.
  • Additionally, we will delve into network controls offered by the cloud provider.
  • You will also get an overview of various methods to securely connect to cloud networks.

Data Protection in the Cloud

  • We will cover techniques and methods related to protecting data stored in the cloud.
  • We will leverage encryption capabilities offered by AWS.
  • Finally, we will demonstrate best practices for encryption key management.

Defensive Security in the Cloud

We will tie this all together by looking at how to incorporate security in your cloud operational model.

  • We will monitor systems to discover risks early on.
  • We will use tools to discover vulnerabilities and insecure configurations.
  • We will also review some of the AWS cloud services which can defend web applications from external threats.
