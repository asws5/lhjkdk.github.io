Leveraging IAM Roles Exercise Solution Summary

In this exercise, you have seen how to reduce the risk associated with an IAM user by creating a role that has elevated permissions.

If the IAM user’s credentials were stolen, or MFA was disabled and someone was able to get access to the IAM user’s account and login, the perpetrator would not be able to take any actions on the account until their MFA device was restored and they assumed the role.
