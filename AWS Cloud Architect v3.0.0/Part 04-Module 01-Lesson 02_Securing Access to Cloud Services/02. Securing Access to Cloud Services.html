Big Picture First Summary

Securing Access

Identity and Access Management in the cloud is the cornerstone of a secure environment.

Securing access to the control plane will determine security on a network, data, and resource consumption level.

Big Picture Heading Image

Big Picture Summary

Key Terms

Authentication

Who can sign in and use the API.

Authorization

What permissions that user has.

Big Picture Quiz

With authenticated access to the AWS API, assuming they had the permissions to do so, a user could potentially:
SOLUTION:
  • Read data in an S3 bucket
  • Change the admin password of an RDS database
  • Open a security group rule to allow SSH access.
  • Retrieve files from an EBS disk volume
