10. Restriction Quizzes

Restrict Access to Cloud Resources Easy Quiz

Identify the true statements below about NACLs and Security groups.
SOLUTION:
  • Security groups do not have explicit deny rules, whereas NACLs can.
  • Security groups can be applied to a specific instance, whereas NACLs are applied to an entire subnet.

Restrict Access Example

Question 2

Your company has provisioned a database instance in AWS. We would like to configure Security Group rules to secure this database.

The following are our connectivity requirements:

  • The database has been launched in a private subnet with a CIDR block of 10.10.1.0/24.
  • The database listens on TCP port 3306.
  • Applications which need to connect to the database are running in a separate subnet with a CIDR block of 10.11.3.0/24.
  • The mentioned application instances have been launched with their own security group sg-123456.
  • There are other servers in the application subnet which should not be able connect to this database.
  • The data engineering team who has a source IP range of 172.16.0.0/8, also needs to be able to connect to the database on port 3306.

Restrict Access to Cloud Resources Hard Quiz

Which of the following security group rule configurations would most effectively restrict inbound traffic to the database?
SOLUTION:
  • Allow TCP - 172.16.0.0/8 - port 3306 Allow TCP - sg-123456 - port 3306
