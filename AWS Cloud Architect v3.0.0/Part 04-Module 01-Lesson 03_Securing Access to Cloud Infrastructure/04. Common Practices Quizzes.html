04. Common Practices Quizzes

Common practices for accessing servers in the cloud Easy Quiz

Which of the following methods for providing access to an instance can increase the risk of unauthorized access (and therefore should be avoided)?
SOLUTION:
  • Use a private key pair and share the private key .pem file with the rest of the developers on the team. Store it on your laptop or Github repo to use whenever you need to login to the instance.
  • Set up password authentication on the instance and send the passwords to appropriate parties.

Common practices for accessing servers in the cloud Hard Quiz

Research the following security vulnerability: CVE-2020-0610

You have launched a Windows server 2019 instance in your AWS environment. Let’s say an attacker had network connectivity to your AWS instance - which of the following outcomes could be expected?
SOLUTION:
  • An attacker may be able to run a script on the server allowing them to copy data from disk to an external location.
  • An attacker might be able to run a command allowing them to make AWS API calls using the instance's role.
