Tips to make your project standout:

  1. Students demonstrate that they can make and apply changes using cloud formation templates instead of through the console.
  2. Students are able to research, identify, and demonstrate defense of application security vulnerabilities (OWASP top 10) using a WAF
  3. Students are able to brainstorm and suggest additional hardening opportunities beyond what is done in the project
